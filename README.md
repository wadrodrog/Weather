# Weather Java API

Simple Java program that gets weather forecast via Open-Meteo API.

This is a learning project.

## Features

- Search for city via interactive text interface
- Get hourly weather forecast

## Tech stack

- Nested classes
- Packages
- JSON parsing org.json
- HTTP client
- Maven project
